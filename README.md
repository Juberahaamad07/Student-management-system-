# Student Management System

## Project Overview

The Student Management System is a Python-based mini project designed to manage student information efficiently. The application provides a simple graphical user interface (GUI) using Tkinter and stores student data in a CSV file.

This project was developed as part of a Python Programming course mini project to improve practical programming skills and understand concepts such as functions, file handling, CSV operations, GUI development, and event-driven programming.

## Objectives

- To create a simple system for managing student information.
- To practice Python programming concepts.
- To understand GUI development using Tkinter.
- To learn how to store and manage data using CSV files.
- To understand basic CRUD operations.
- To improve practical programming and problem-solving skills.

## Features

- Add Student
- Search Student
- Update Student
- Delete Student
- View All Students
- Clear Input Fields
- Store student data using CSV files
